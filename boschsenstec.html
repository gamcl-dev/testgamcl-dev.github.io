Bosch Senstec
Internet of things 3D graphical interface

Working with Bosch I created a virtual environment to visualise the realtime data from the sensors and allow two-way interactions.

Some points about Bosch Senstec Project

  • Developed for CES2014 Showcase
  • Working remotely with Bosch in Germany
  • Developed with Unity, C#, MySql
  • Cross platform development; Mobile, Web and Desktop

Sensors relayed the information which the user could visualise in a 3D representation of the CES2014 floor space. This included the control of doors, lights, air conditioning, heating information, control of robotics.
